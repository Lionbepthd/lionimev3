# Lionime

Lionime adalah website pribadi untuk menonton anime secara online. Website ini dibangun dengan tujuan menyediakan antarmuka yang nyaman untuk mencari dan menonton anime, dengan integrasi dari berbagai sumber data.

## Fitur

*   **Jelajahi Anime:** Cari anime berdasarkan judul.
*   **Kategori:**
    *   **Home:** Menampilkan anime sedang tayang, anime terbaru, dan anime populer.
    *   **List:** Daftar lengkap semua anime.
    *   **Genre:** Filter anime berdasarkan genre.
    *   **Movie:** Daftar anime movie.
    *   **Jadwal:** Jadwal rilis anime terbaru.
*   **Detail Anime:** Informasi lengkap tentang anime, termasuk sinopsis, rating, dan genre.
*   **Streaming:**
    *   **Link Legal:** Menyediakan link streaming legal dari berbagai platform (melalui API Jikan).
    *   **Link Alternatif:** Menyediakan link streaming dari sumber lain (melalui API SankaVollerei).

## Teknologi

*   **HTML5**
*   **CSS3**
*   **JavaScript (Client-side)**
*   **API:**
    *   [Jikan API v4](https://docs.api.jikan.moe/) - Untuk informasi dasar anime dan link streaming legal.
    *   [SankaVollerei Anime API](https://www.sankavollerei.com/anime/) - Untuk data anime, jadwal, dan link streaming alternatif dari Otakudesu.
